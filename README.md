# Jordyn Bowe — 1.6 Exercise

## Portfolio

A personal portfolio page built with Next.js and Tailwind CSS. Displays a profile card, about me section, tech stack, and skills.

## Tech

- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [tech-stack-icons](https://www.npmjs.com/package/tech-stack-icons)

## Getting Started

```bash
cd my-app
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
